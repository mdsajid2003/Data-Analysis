|------|-------------|

---

## 📊 Project Overview

This project analyzes Citi Bike trip data to uncover trends and patterns in urban bike-sharing usage. The analysis aims to provide actionable insights into:

- **Ridership trends** — peak usage hours, days, and seasons
- **Station performance** — most and least popular start/end stations
- **User behavior** — trip duration, frequency, and subscriber vs. casual rider comparisons
- **Operational insights** — data-driven recommendations for fleet and station management
